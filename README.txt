屏幕抓取与推流工具使用说明

功能介绍:
1. 屏幕抓取: 可以抓取整个屏幕或选择特定区域进行抓取
2. 实时预览: 抓取的内容会实时显示在预览窗口中
3. 视频推流: 支持将抓取的内容实时推流到RTMP服务器

使用方法:
1. 运行应用程序
2. 点击"选择区域"按钮可以选择特定的屏幕区域进行抓取
   (如果不选择区域，则默认抓取整个屏幕)
3. 点击"开始抓取屏幕"按钮开始抓取
4. 在推流地址输入框中输入RTSP推流地址
5. 点击"开始推流"按钮开始推流
6. 点击"停止推流"按钮停止推流
7. 点击"停止抓取"按钮停止抓取屏幕

依赖项:
本工具需要FFmpeg来进行视频推流。请确保您已安装FFmpeg并配置正确:

安装方法:
1. 从FFmpeg官方网站(https://ffmpeg.org/)下载适合您系统的版本
2. 将FFmpeg的可执行文件(ffmpeg.exe)放在以下任一位置:
   a. 与本应用程序同一目录下
   b. 系统PATH环境变量中的任何目录

配置RTMP服务器:
如果您需要测试推流功能，可以使用Nginx+RTMP模块搭建本地RTMP服务器，
或使用第三方直播平台提供的RTSP推流地址。

注意事项:
1. 确保您的系统已安装.NET 9.0或更高版本
2. 推流质量取决于您的网络带宽和服务器性能
3. 长时间抓取屏幕可能会增加系统资源消耗
